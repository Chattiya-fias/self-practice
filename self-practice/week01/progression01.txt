- part01 ทบทวนการสร้าง array และลองเก็บค่าหลาย type เอาไว้ 
ทบทวนการใช้ loop เพื่อแสดงค่าแต่ละตัว และเพิ่มเงื่อนไขให้มีการเช็คตัวแปรก่อนว่าเป็น array หรือไม่
- part02 ทบทวนและฝึกการยืดหดของ array ซึ่งเนื่องจาก array มีคุณสมบัติที่หดได้
แต่ในคาบเรียนไม่มีการยกตัวอย่างการ shrinking เลยค้นหา method ที่จะทำให้หดได้มาฝึกทำ
- part03 ฝึกการเขียนแสดงผลค่าที่ต้องการจาก array ที่ซับซ้อน
- part04 ทบทวนการใช้ spread operator และฝึกการใช้ร่วมกับ method อื่นเช่น reverse
ทบทวนการรวม array ด้วย spread การแทรกด้วย spread และการ copy ด้วย spread
