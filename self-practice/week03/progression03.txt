- part 01 เรียนรู้ซ้ำถึงสร้าง function ทั้ง 3 แบบ และเนื่องจากยังไม่คล่อง arrow function
จึงนำโจทย์เก่าของ self-practce 01 ที่เขียนเป็นแบบ function declaration มาลองเขียนเป็น arrow function

- part 02 เรียนรู้เรื่อง higher-order function จากในสไลด์ซ้ำอีกครั้งถึง concept ของ higher-order function และฝึกทำโจทย์ที่ใช้ chatgpt ช่วยคิดให้

- part 03 เรียนเรื่อง parameter passing ซ้ำอีกครั้ง ทำให้เห็นและเข้าใจการทำงานของ parameter ใน function ทั้ง 2 แบบ
และเข้าใจ pass by value กับ pass by reference มากขึ้น

- part 04 ตอนเรียนในห้องเรื่อง argument object รู้สึกว่ามันทำงานคล้าย index ของ function เลยศึกษาเพิ่มเติม จึงรู้ว่ามัน 
array-like object และเนื่องจากในห้อง อาจารย์ได้ตั้งโจทย์ให้หา sum โดย parameter ไม่จำกัดจำนวนตัวและรองรับทุกรูปแบบ
จริงๆเห็นเพื่อนส่วนใหญ่ใช้เป็น sum += argu แต่หนูมองว่าเนื่องจากต้องรองรับทุกตัวหากเป็น string ก็จะเอาไป sum ไม่ได้
พอได้รู้จัก ฟังก์ชัน Number() ของ JavaScript เลยเอามาใช้ให้แปลง argu เป็นตัวเลขก่อนถ้าแปลงไม่ได้ก็ค่อยเปลี่ยนเป็น 0 ถึงจะรองรับได้ทุกรูปแบบ

- part 05 ทำโจทย์ practice ที่อาจารย์ให้ในห้อง ตอนที่ทำในห้องทำถึงแค่แปลง string เป็น array แล้ว
ขึ้น for loop เพื่อลูปเอา index ใน array มากำหนดเป็น key ใน object และสร้างตัวแปรที่เก็บ object ว่างๆ
แต่ในส่วนหลังจากนี้ไม่รู้วิธีที่จะเอา index ใส่เข้าไปเพื่อให้เป็น key จึงค้นหาตัวอย่างหลากหลายวิธีจาก chatgpt
แล้วเลือกวิธีที่สอดคล้องกับแนวทางการคิดของตัวเองและเข้าใจได้ง่ายมาปรับใช้ 
และได้รู้เพิ่มเติมว่า object[x] เป็นการกำหนด key ส่วน object[x] = xxx เป็นการกำหนด value
